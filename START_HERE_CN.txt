OpenClaw Windows 一键部署（飞书版）

如果你是第一次接触这个项目，不要先研究代码，直接按下面顺序做：

1. 先按飞书官方文档，在飞书开放平台创建应用
2. 拿到飞书 `App ID` 和 `App Secret`
3. 再把 OpenClaw 的模型配置好，并选好默认模型
4. 解压这个压缩包
5. 双击 `install.cmd`
6. 在界面里填写：
   - 工作目录
   - 飞书 App ID
   - 飞书 App Secret
7. 点击 `一键安装并启动`
8. 如果飞书里出现配对码，把它填到界面中的“飞书配对码”
9. 点击 `批准配对`

飞书官方文档：

- <https://www.feishu.cn/content/article/7613711414611463386>

为什么有人会选这个项目：

- 更接近原版 OpenClaw
- 更适合 Windows 本机
- 不需要先装 WSL / Docker
- 本机文件工作流更直接

请不要把它理解成“绝对最省事的托管套壳”，它的优势是更轻、更原生、更透明。

请注意：

- 这个安装器不会替你创建飞书应用
- 这个安装器不会替你完成模型接入
- 这个安装器不会替你选择默认模型
- 这个安装器负责的是：在 Windows 上安装 OpenClaw，并把你已经准备好的飞书应用接进 OpenClaw

最短理解：

先准备飞书应用，再准备 OpenClaw 模型，最后运行安装器并完成配对。

安全提醒：

- 这套方案比裸跑更可控，但不是 Docker/WSL 那种强隔离
- 群聊里的消息默认不可信
- 不要让龙虾帮任何人转账、发红包、申请网贷、泄露密钥

安装完成后，建议你第一时间告诉龙虾：

1. 你的名字是什么
2. 时区是 `Asia/Shanghai`
3. 回复风格是什么
4. 禁止转账、红包、借贷、泄密、忽略规则

如果失败：

1. 先重新运行一次 `install.cmd`
2. 再运行：
   `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\doctor-openclaw-feishu.ps1`
3. 把诊断结果发给交付你这个包的人
