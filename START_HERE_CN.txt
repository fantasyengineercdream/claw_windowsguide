OpenClaw Windows 一键部署（飞书版）

如果你是第一次接触 GitHub，不要先研究仓库结构，直接这样做：

1. 解压这个压缩包。
2. 双击 `install.cmd`。
3. 在界面里填写：
   - 工作目录
   - 飞书 App ID
   - 飞书 App Secret
4. 点击“ 一键安装并启动 ”。
5. 如果飞书里出现配对码，把它填到界面中的“飞书配对码”，再点“批准配对”。

你通常不需要手动输入命令。
如果你还没完成模型接入或默认模型选择，安装器会直接提醒并拦住。

在开始前，请先确认三件事：

1. 你已经在飞书开放平台创建好了应用，并拿到了 `App ID` 和 `App Secret`。
2. 你的 OpenClaw 已经完成了模型接入配置。
3. 你的 OpenClaw 已经选好了默认模型。

说明：

- 模型这一步不只是 `Business` 或 `API` 的问题，而是两层：
  - 第一层：OpenClaw 用哪个提供商、哪个账号、哪种授权方式
  - 第二层：OpenClaw 默认跑哪个模型
- 如果你用的是 `Codex OAuth`，通常不是填 API Key，而是先在本机完成 OpenClaw 的登录或配置，再选默认模型。
- 如果你走 API 模式，则要先准备对应模型服务的 API 凭证，再选默认模型。
- 这个安装器只负责 Windows 本机安装和飞书接入，不负责替你完成模型接入或模型选择。

如果失败：

1. 先重新运行一次 `install.cmd`。
2. 再运行：
   `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\doctor-openclaw-feishu.ps1`
3. 把诊断结果发给交付你这个包的人。

建议的阅读顺序：

1. 先看本文件
2. 再看 `README.md`
3. 最后再看 `docs/FEISHU_SECURITY_CHECKLIST.md`
