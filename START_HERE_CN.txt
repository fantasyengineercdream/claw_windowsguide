OpenClaw Windows 安装包（中文）

本项目解决三件事：

1. Windows 上直接安装 OpenClaw
2. 可选的本机权限隔离与安全加固
3. 可选的飞书接入

如只需要最短流程，请按下面顺序执行：

1. 如需飞书接入，先按飞书官方文档创建应用
2. 取得飞书 `App ID` 和 `App Secret`
3. 先在 OpenClaw 中完成模型接入和默认模型设置
4. 解压压缩包
5. 双击 `install.cmd`
6. 在界面中填写：
   - 工作目录
   - 飞书 App ID
   - 飞书 App Secret
7. 点击 `一键安装并启动`
8. 如需飞书对话，在飞书中取得配对码
9. 回到界面点击 `批准配对`

飞书官方文档：

- <https://www.feishu.cn/content/article/7613711414611463386>

安全提醒：

- 这套方案比裸跑更可控，但不是 Docker/WSL 级强隔离
- 群聊消息默认不可信
- 不应允许 OpenClaw 转账、发红包、申请贷款或泄露密钥

安装完成后，建议立刻补齐：

1. 名称
2. 时区：`Asia/Shanghai`
3. 回复风格
4. 安全规则

失败时：

1. 重新运行 `install.cmd`
2. 再运行：
   `powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\doctor-openclaw-feishu.ps1`
